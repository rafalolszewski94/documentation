---
title: 'Get Universal IP Address'
openapi: 'GET /api/v1/ip64'
---

Returns the client's public IP address (IPv4 or IPv6, whichever is detected). This is the universal endpoint that works with both IP versions.

## Parameters

<ParamField query="format" type="string" default="text">
  Response format: 'text' for plain IP string, 'json' for JSON object with geolocation data, 'jsonp' for JSONP callback
</ParamField>

<ParamField query="callback" type="string" default="callback">
  Callback function name for JSONP responses. Only used when format=jsonp.
</ParamField>

## Response Examples

### Plain Text Response

```bash
curl https://api.ipbun.com/api/v1/ip64
```

**IPv4 Example:**
```text
203.0.113.1
```

**IPv6 Example:**
```text
2001:db8::1
```

### JSON Response with Geolocation

```bash
curl "https://api.ipbun.com/api/v1/ip64?format=json"
```

**IPv4 with Geolocation:**
```json
{
  "ip": "203.0.113.1",
  "geolocation": {
    "country": "United States",
    "countryCode": "US",
    "city": "New York",
    "region": "New York",
    "latitude": 40.7128,
    "longitude": -74.0060,
    "timezone": "America/New_York",
    "isp": "Example ISP LLC",
    "org": "Example Organization",
    "as": "AS12345",
    "asname": "EXAMPLE-AS",
    "mobile": false,
    "proxy": false,
    "hosting": false
  }
}
```

**IPv6 with Geolocation:**
```json
{
  "ip": "2001:db8::1",
  "geolocation": {
    "country": "Japan",
    "countryCode": "JP",
    "city": "Tokyo",
    "region": "Tokyo",
    "latitude": 35.6762,
    "longitude": 139.6503,

    "as": "AS12345",
    "asname": "EXAMPLE-AS",
    "mobile": false,

  }
}
```

**IP-only Response (when geolocation unavailable):**
```json
{
  "ip": "192.0.2.1"
}
```

### JSONP Response

```bash
curl "https://api.ipbun.com/api/v1/ip64?format=jsonp&callback=myCallback"
```

**IPv4 JSONP:**
```javascript
myCallback({"ip":"203.0.113.1"})
```

**IPv6 JSONP:**
```javascript
myCallback({"ip":"2001:db8::1"})
```
