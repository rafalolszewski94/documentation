## Welcome to IPBun

IPBun is a simple, fast, and reliable public IP address API built with Bun. Get your public IP address in multiple formats with optional geolocation data.

## Quick Start

Get your IP address with a simple API call.

```bash
curl https://api.ipbun.com/api/v1/ip
# Returns: 192.168.1.1
```

## Features

What makes IPBun reliable and easy to use.

- **Fast Response Times** - Built with Bun for lightning-fast performance
- **Multiple Formats** - Text, JSON, and JSONP support
- **Geolocation Data** - Optional location information with coordinates
- **Rate Limiting** - Fair usage with clear limits and headers
- **IPv4 & IPv6 Support** - Works with both address types
- **OpenAPI Specification** - Complete API documentation
