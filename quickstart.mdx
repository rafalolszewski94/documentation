## Get your IP address instantly

Start using IPBun API with these simple steps.

### Step 1: Make your first API call

Get your public IP address with a simple curl command:

<CodeGroup>
```bash cURL
curl https://api.ipbun.com/api/v1/ip
# Returns: 192.168.1.1
```

```javascript JavaScript
fetch('https://api.ipbun.com/api/v1/ip')
  .then(response => response.text())
  .then(ip => console.log(ip));
```

```python Python
import requests
response = requests.get('https://api.ipbun.com/api/v1/ip')
print(response.text)
```
</CodeGroup>

### Step 2: Try different response formats

IPBun supports multiple response formats:

<CodeGroup>
```bash Plain Text (default)
curl https://api.ipbun.com/api/v1/ip
# Returns: 192.168.1.1
```

```bash JSON with geolocation
curl "https://api.ipbun.com/api/v1/ip?format=json"
```

```json JSON Response
{
  "ip": "192.168.1.1",
  "type": "ipv4",
  "geolocation": {
    "country": "United States",
    "country_code": "US",
    "city": "New York",
    "coordinates": {
      "latitude": 40.7128,
      "longitude": -74.0060
    }
  }
}
```
</CodeGroup>

### Step 3: Choose your endpoint

Select the right endpoint for your needs:

| Endpoint | Purpose | Example Response |
|----------|---------|------------------|
| `/api/v1/ip` | IPv4 address only | `192.168.1.1` |
| `/api/v1/ip6` | IPv6 address only | `2001:db8::1` |
| `/api/v1/ip64` | Universal (IPv4/IPv6) | `192.168.1.1` or `2001:db8::1` |
| `/api/v1/health` | Service health check | `{"status": "healthy"}` |
