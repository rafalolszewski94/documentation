---
title: 'Get IPv4 Address'
openapi: 'GET /api/v1/ip'
---

Returns the client's public IPv4 address. This endpoint specifically returns IPv4 addresses and will return an error if no IPv4 address is detected.

## Examples

### Plain Text Response

```bash
curl https://api.ipbun.com/api/v1/ip
```

```text
203.0.113.1
```

### JSON Response with Geolocation

```bash
curl "https://api.ipbun.com/api/v1/ip?format=json"
```
